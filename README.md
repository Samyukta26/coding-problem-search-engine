## Objective
Build a search engine that helps users find coding problems from multiple platforms using TF-IDF based ranking.

## Features
- Keyword Search
- TF-IDF Ranking
- Difficulty Filter
- Platform Filter
- Topic Tags
- Favorites
- Search History

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Dataset: JSON
- Search Algorithm: TF-IDF
